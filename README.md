# Ciencia-de-Datos---BMW-Modelo-predictivo-ventas-2025-en-adelante.
Este proyecto corresponde a un análisis de datos enfocado en el modelado predictivo de la columna Sales Classification, utilizando información histórica de ventas de BMW entre 2010 y 2024. La carpeta src contiene todos los scripts funcionales del pipeline: un archivo para validar la presencia de nulos, vacíos y duplicados; un preprocess encargado de limpiar el dataset cuando se detectan dichos problemas; un train_model para el entrenamiento del modelo predictivo; y un evaluate para el testeo del rendimiento. En la carpeta models se almacena el modelo final entrenado, mientras que en notebooks se encuentran cuadernos descriptivos que documentan tanto el análisis exploratorio como la comparación entre valores reales y predichos, además de un notebook asociado al dataset original sin la columna Sales_Volume.
